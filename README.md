# Agente Gamer Responsável

Projeto acadêmico desenvolvido em Python + CustomTkinter.

## Funcionalidades
- Sistema baseado em palavras-chave
- Regras éticas
- Segurança digital
- Registro de logs
- Interface gráfica

## Tecnologias
- Python
- CustomTkinter

## Como executar

pip install -r requirements.txt

python app_agente_gamer_responsavel.py
